# Product Documentation

This directory records the product intent for the OpenCode harness agents plugin bundle.

## Files

| File | Purpose | Read when |
| ---- | ------- | --------- |
| [`personas.md`](personas.md) | Users, maintainers, reviewers, and future agents served by the bundle. | You need to understand whose workflow a change supports. |
| [`user-journeys.md`](user-journeys.md) | Important end-to-end workflows for local development and OpenCode usage. | You are changing setup, plugin loading, bundled agents, tools, or commands. |
| [`acceptance-criteria.md`](acceptance-criteria.md) | Product-level done rules and harness-engineering success criteria. | You are deciding whether a feature or repo change is complete. |
